Je conçois et livre des **produits SaaS de qualité production** — du schéma de base de données au pipeline de déploiement.

## Principes d'Architecture

```
✦  Domain-Driven Design (DDD)        ✦  Méthodologie 12-Factor App
✦  API-first, mobile-ready           ✦  JWT (15min) + Refresh tokens (7j)
✦  Patterns SaaS multi-tenant        ✦  CI/CD sur chaque projet
✦  Soft delete + pistes d'audit      ✦  OWASP Top 10 — systématique

```
